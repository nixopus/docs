The Nixopus extension brings the full deploy experience into VS Code and Cursor. An AI-powered agent lives in your sidebar — it analyzes your codebase, generates configs, and deploys your app without you ever leaving the editor.

## Key features

- **Chat interface** — Talk to the deploy agent in natural language. Ask it to deploy, diagnose, or explain.
- **Codebase analysis** — The agent reads your project structure, detects frameworks and services, and generates the right configuration.
- **Dockerfile generation** — No Dockerfile? The agent creates one tailored to your stack.
- **One-click deploy** — Go from code to live in a single conversation.
- **Workspace sync** — Link your editor workspace to a Nixopus project for automatic context.
