Extensions add capabilities to Nixopus — monitoring integrations, backup tools, custom workflows, and more. Browse and install them from the built-in marketplace.

## The marketplace

Navigate to **Extensions** from the sidebar footer. You'll see a grid of available extensions with search, sort, and category filters.

Extensions add functionality beyond core deployments — monitoring integrations, backup tools, custom workflows, and more.

## Installing an extension

<Steps>
  <Step title="Browse or search">
    Find the extension you want from the marketplace grid. Each listing shows a description and category.
  </Step>
  <Step title="Install">
    Click on an extension to view its detail page. Click **Install** to add it to your organization.
  </Step>
  <Step title="Configure">
    Some extensions require configuration (API keys, target apps, schedules). Fill in the required fields from the extension's overview tab.
  </Step>
</Steps>

## Running extensions

From the extension detail page, click **Run** to execute it. You can also **Fork** an extension to create your own customized version.

## Execution history

The extension detail page shows a history of past runs with start/end time, status, and output.

<Note>
  Extensions run in isolated containers on your deployment machine. They have access only to the permissions granted during installation.
</Note>
